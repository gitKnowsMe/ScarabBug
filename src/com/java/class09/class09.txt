


                Increment/Decrement Operator ( ++ , --) : It'll increment or decrement by 1

      ++ : Increment the value by 1
      -- : Decrement the value by 1

Postfix Operator
      exp++;
          * operators comes after the variable
          * first complete all other operators and then increment the value of the variables

Prefix Operator
      ++exp;
          * operators comes before the variable
          * first complete all other operators and then increment the value of variable
